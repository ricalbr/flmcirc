# FLMcirc

Python utility class for the design of Integrated Optical Circuits with
possibility of output G-Code files.

### Requirements

- `matplotlib`
- `numpy`
- `scipy`

### Install

```
mkdir circuitDesign && cd circuitDesign
git clone https://github.com/ricalbr/flmcirc
```
